# interview-quiz

## Setup

### Local Setup

#### server

```bash
cd server
npm install
npm run start:dev
```

#### client

```bash
cd client
npm install
npm run start:dev
```

### Docker Setup

```bash
 docker run --env-file .env -p 8080:8080 interview-quiz
```
